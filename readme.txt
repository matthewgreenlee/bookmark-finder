Bookmark Finder is a chrome extension which help you find the bookmark from within your local bookmarks with the keywords you type in the omnibox.

Usage:
1. type in "bf" in the omnibox followed by pressing "Tab" button to invoke Bookmark Finder.
2. type in the keyword you want to search for bookmark.
3. Bookmark Finder will help you find bookmarks from your local bookmarks in the real time with the above keyword.
4. simple pressing "Enter" to see all the bookmarks related to the inputed keyword.
5. move down or up to select the highlighted bookmark and press "Enter" to access the bookmarked website.
